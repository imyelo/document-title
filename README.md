# @yelo/document-title
> Set the document title

inspired by [mishe](https://github.com/mishe/blog/issues/57)


## Install
```
npm i --save @yelo/document-title
```


## Usage
```
var title = require('@yelo/document-title');

title('Hey Wechat');
```


## API
### title.source(src)
change the iframe src
```
title.source('/test.png');
```
